📞 SOPORTE RÁPIDO:
═══════════════════════════════════════════════════════════════════════════════

Problema: DATABASE_URL no encontrada
  Solución: Vercel → Settings → Environment Variables → Agregar DATABASE_URL

Problema: Connection refused en Railway
  Solución: Railway → Postgres → Status (debe estar Ready)

Problema: Table already exists
  Solución: Normal, migraciones usan IF NOT EXISTS. Continúa.

Problema: 401 Unauthorized en API
  Solución: Token JWT inválido. Loguéate en app para obtener token fresco.

Problema: Datos aún no persisten
  Solución: Verifica que DATABASE_URL está en Vercel y redeploy completó.
