# Sección 13 — Inventarios (NIIF para PYMES)

## Alcance
Aplica a todos los inventarios excepto:
- Obra en proceso de contratos de construcción (Sección 23)
- Instrumentos financieros (Sección 11/12)
- Activos biológicos (Sección 34)

## Medición de los Inventarios
Los inventarios se medirán al **menor de**:
- Costo
- Valor neto realizable (VNR)

## Costo de los Inventarios

### Componentes del Costo
- Costos de adquisición (precio de compra + aranceles + transporte)
- Costos de transformación (mano de obra directa + costos indirectos)
- Otros costos incurridos para darles su condición actual

### Costos Excluidos del Inventario
- Desperdicios anormales
- Costos de almacenamiento (salvo proceso productivo)
- Costos administrativos indirectos
- Costos de venta

### Fórmulas de Costo Permitidas

| Método | Descripción | Aplicación |
|--------|-------------|------------|
| FIFO (PEPS) | Primeras entradas, primeras salidas | Más común |
| Costo promedio ponderado | Promedio de todos los costos | Alternativa |
| Identificación específica | Para inventarios no intercambiables | Artículos únicos |

*Nota: NIIF PYMES NO permite el método LIFO (UEPS).*

## Valor Neto Realizable
VNR = Precio estimado de venta
- Costos estimados de terminación
- Costos estimados de venta

## Reconocimiento como Gasto
Los inventarios se reconocen como gasto cuando se venden:
- El importe en libros se reconoce como costo de ventas
- Cualquier deterioro se reconoce en el período en que ocurre
- Las reversiones de deterioro se reconocen como reducción del gasto

## Deterioro de Inventarios
Reconocer cuando el VNR < Costo. Causas comunes:
- Mercancía dañada
- Total o parcialmente obsoleta
- Precios de venta disminuidos
- Costos de terminación o venta aumentados

## Revelaciones Requeridas
- Políticas contables adoptadas para medir inventarios
- Importe en libros por clasificación
- Importe reconocido como gasto
- Importe de deterioros reconocidos

## Alertas de Auditoría
- Verificar uso de método FIFO o promedio (no LIFO)
- Revisar que el VNR se calcule correctamente
- Identificar inventarios obsoletos o de baja rotación
- Confirmar que los costos indirectos fijos se asignan correctamente
- Evaluar corte de inventarios en el conteo físico
